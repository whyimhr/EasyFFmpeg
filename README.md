# EasyFFmpeg

A lightweight macOS GUI wrapper for FFmpeg written in Swift.

## Features

- Simple interface for common FFmpeg tasks
- Fast video/audio conversion
- Native macOS experience
- No command-line required

## Download

👉 Download the latest version from **Releases**

## Installation

1. Download the `.dmg` from Releases
2. Open the `.dmg`
3. Drag **EasyFFmpeg.app** into **Applications**
4. Launch the app

## If macOS blocks the app

Run in Terminal:

```bash
xattr -rd com.apple.quarantine /Applications/EasyFFmpeg.app
